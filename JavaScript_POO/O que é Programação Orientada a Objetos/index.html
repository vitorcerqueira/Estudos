O que é Programação Orientada a Objetos

A Programação Orientada a Objetos (POO) é um paradigma de programação que se baseia na ideia de modelar sistemas de software usando objetos. Um objeto é uma instância de uma classe, que é uma estrutura que define o comportamento e as propriedades dos objetos. POO é amplamente utilizada em várias linguagens de programação, incluindo JavaScript, mas HTML e CSS não são linguagens orientadas a objetos. Vou fornecer um exemplo em JavaScript para ilustrar a POO.
