Estruturas de Repetição While

Estruturas de repetição servem para executar um mesmo bloco de codigos varias vezes

o While serve para criar um laço de repetição que sera executado enquanto uma determinada condição for verdadeira.

o While e todas as outas estruras de repetição que veremos aceitam o break para parar a repetição a qualquer momento

é importante que sempre exista uma forma de condição se tornar falsa, ou então o bloco ficara se repetindo para sempre

ENQUANTO condição FAÇA algo varias vezes

sintaxe:
while(condicao){
//...}
