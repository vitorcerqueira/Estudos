Estruturas Condicionais Switch

Permite criar blocos condicionais como IF, porem com uma estrutura que avalia uma expressao que avalia uma expressao e testa o resultado contra varios casaso executando os casos correspontendes é possivel definir um caso padrao que sempre é executado, tambem é possivel parar a execução apos algum dos casos.
SWITCH expressão
CASOa:faça uma coisa.
CASOb:faça outra coisa e PARE.
CASOc:faça outra coisa.
POR PADRÃO:faça algo sempre.

Sintaxe:
switch(expressão){
case "a":
//...
case "b":
//...
break
case "c":
//...
default:
//...
}
