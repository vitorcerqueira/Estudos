Elementos Semânticos no HTML 5

O que é semântica?

É o estudo do significado das palavras e frases em um idioma. Assim, em HTML, elementos semânticos são elementos com significado.

Tabela de Elementos
Tag Descrição Aplicações
<article> Define um conteúdo independente, que faça sentido em si mesmo e possa ser destacado do restante do site. Postagens em blog, artigos, comentários de usuários, cartões de produtos.
<aside> Define um conteúdo a ser mostrado na parte lateral. Deve estar associado indiretamente ao conteúdo que o cerca. Etiquetas, quadros e comentários postos ao lado de um conteúdo principal.
<details> Define conteúdo adicional que o usuário poderá mostrar ou ocultar. Cria um widget interativo, oculto por padrão, que poderá ser mostrado por um clique do usuário que deseje ler os detalhes ali contidos.
<figcaption> Define uma legenda para uma figura que será mostrada utilizando a tag <figure>. A legenda aqui contida pode ser colocada antes ou após uma figura que se queira explicar.
<figure> Especifica que uma imagem será inserida no documento por meio do atributo "img". Colocação de figuras, fotografias, diagramas ou qualquer outro arquivo de imagem que se queira exibir.
<footer> Define o rodapé de um documento ou uma seção. Poderá trazer dados de autoria, dados de contato, mapa do site, links de navegação para o topo, documentos relacionados etc.
<header> Especifica o cabeçalho de um documento ou seção. Poderá trazer contêineres de elementos introdutórios, um conjunto de links navegáveis, um ou mais elementos de titulação (<h1> - <h6>), o logo ou ícone da pagina e informações de autoria.
<main> Especifica o conteúdo principal do documento. O conteúdo dentro deste elemento deve ser único ao documento, não podendo trazer conteúdo que seja repetido ao longo do documento como barras laterais, links de navegação, informações de autoria, logos, e formulários de pesquisa. O elemento <main> NÃO pode descender dos elementos <article>, <aside>, <footer>, <header>, ou <nav>.
<mark> Estabelece marcação para o texto. A palavra ou texto dentro deste elemento aparecerá em highlight.
<nav> Define links de navegação, que serão abaixo aninhados usando o atributo <a href="">. Nem todos os links de um documento devem estar contidos neste elemento, pois ele serve para marcar uma seção inteira de links de navegação.
<section> Define uma seção no documento. Separará as seções do documento de maneira a garantir a organização das mesmas dentro do código.
<summary> Define um cabeçalho para o elemento <details>. O usuário poderá clicar no cabeçalho para exibir o seu conteúdo. Este elemento deve vir sempre ANTES do elemento <details>.
<time> Especifica uma data ou abre o atributo "datetime". Caso se utilize o atributo "datetime", a data será traduzida pelo navegador em um formato compreensível pela máquina, possibilitando que seja ofertado ao usuário um lembrete a ser adicionado no calendário. Também poderá ser utilizado por mecanismo de busca para realizar pesquisas inteligentes.

A documentação completa sobre os elementos semânticos poderá ser encontrada em W3Schools.
