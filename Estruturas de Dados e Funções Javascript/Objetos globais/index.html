Objetos globais

Em JavaScript, existem diversos objetos globais que fornecem funcionalidades essenciais e recursos pré-construídos. Esses objetos estão disponíveis em qualquer contexto de código no navegador ou no ambiente Node.js. Alguns dos objetos globais mais comuns incluem:

  • Object: O construtor Object é a base para todos os outros objetos em JavaScript. Ele possui métodos e propriedades úteis para manipular objetos.
  • Array: O objeto Array é usado para armazenar e manipular listas de elementos. Ele oferece métodos para adicionar, remover, filtrar, mapear e percorrer elementos em uma lista.
  • Function: Funções em JavaScript são objetos de primeira classe, o que significa que elas podem ser atribuídas a variáveis, passadas como argumentos e retornadas de outras funções.
  • String: O objeto String é usado para manipular sequências de caracteres. Ele oferece métodos para realizar operações de busca, substituição, concatenação e transformação em strings.
  • Number: O objeto Number é usado para trabalhar com valores numéricos. Ele fornece métodos para realizar operações matemáticas e conversões entre diferentes representações numéricas.
  • Boolean: O objeto Boolean representa um valor booleano, ou seja, verdadeiro (true) ou falso (false). Ele é útil para realizar operações lógicas.
  • Math: O objeto Math fornece funções matemáticas comuns, como cálculos de seno, cosseno, raiz quadrada, potência, arredondamento e muito mais.
  • Date: O objeto Date é usado para lidar com datas e horários. Ele permite a criação, manipulação e formatação de datas.
  • RegExp: O objeto RegExp é usado para trabalhar com expressões regulares, que são padrões utilizados para busca e manipulação de strings com base em regras específicas.
  • JSON: O objeto JSON oferece métodos para transformar objetos JavaScript em formato JSON (JavaScript Object Notation) e vice-versa. É comumente usado para intercâmbio de dados entre um servidor e um cliente.
  • console: O objeto console fornece métodos para exibir informações no console do navegador ou do ambiente Node.js. É frequentemente usado para depuração e registro de informações.
