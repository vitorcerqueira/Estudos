Objetos

é uma estrutura do javascript para trabalhar com dicionarios chave-valor.

O par chave-valor em um objeto javascript é chamado de propriedade.

Diferente dos arrays, seus elementos nao possuem nenhuma sequecia

Uma propriedade tambem pode ter strings, numeros e boleanos como chave

Uma propriedade pode armazenar qualquer tipo de dado como valor

Pode ser criado atraves de chaves{} let objeto = {}

Suas propriedade e funçoes podem ser referencias por encadeamento como ponto "." ou colchetes[]
