Template Literals

Template literals, também conhecidos como template strings, são uma característica da linguagem de programação JavaScript que permite criar strings de forma mais flexível e legível, especialmente quando você precisa incorporar valores de variáveis ou expressões em uma string.

Em vez de usar as aspas simples ('') ou aspas duplas ("") para definir uma string como normalmente faria em JavaScript, você pode usar o acento grave ( ) para criar um template literal. Aqui está um exemplo simples:

const nome = 'Vitor';
const idade = 29;

const mensagem = `Olá, meu nome é ${nome} e eu tenho ${idade} anos.`;
console.log(mensagem);
