Arrow functions

Arrow functions, também conhecidas como "funções de seta" em português, são uma característica introduzida no JavaScript para criar funções de forma mais concisa e simplificada. Elas foram adicionadas ao ECMAScript 6 (também conhecido como ES6) e oferecem uma sintaxe mais curta em comparação com as funções tradicionais (também chamadas de funções anônimas ou funções expressas).

Aqui está um exemplo de uma função tradicional em JavaScript que adiciona dois números

function soma(a, b) {
return a + b;
}
