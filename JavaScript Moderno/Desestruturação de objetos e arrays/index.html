Desestruturação de objetos e arrays

A desestruturação de objetos e arrays é uma característica introduzida no JavaScript (ECMAScript 6 ou ES6) que permite extrair valores de objetos e arrays de forma mais concisa e conveniente, atribuindo-os a variáveis individuais. Isso simplifica a manipulação de dados em estruturas complexas, tornando o código mais legível e menos verboso.

Aqui estão os conceitos básicos da desestruturação de objetos e arrays:

Desestruturação de Objetos:

Desestruturação de objetos simples: Suponha que você tenha um objeto com várias propriedades e deseja atribuir essas propriedades a variáveis individuais. Você pode fazer isso usando a desestruturação de objetos da seguinte maneira:

const pessoa = { nome: 'Alice', idade: 30 };
const { nome, idade } = pessoa;
console.log(nome); // "Alice"
console.log(idade); // 30

Alias de propriedades:

Você também pode atribuir propriedades a variáveis com nomes diferentes usando a sintaxe de alias:
const pessoa = { nome: 'Bob', idade: 25 };

const { nome: nomeDaPessoa, idade: idadeDaPessoa } = pessoa;

console.log(nomeDaPessoa); // "Bob"
console.log(idadeDaPessoa); // 25
