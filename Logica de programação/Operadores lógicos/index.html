Operadores lógicos


TABELA VERDADE - E (AND)
Operador &&
só é verdadeiro se ambas as partes forem verdadeiras
p q p^q
T T T
T F F
F T F
F F F

True && True === True
True && False === False
Falso && True === False
Falso && False === False

TABELA VERDADE - Ou (OR)
Operador ||
só é falso se ambas as partes forem falsas
p q pvq
T T T
T F T
F T T
F F F

True || True === True
True || False === True
Falso || True === True
Falso || False === False

TABELA VERDADE - Não (NOT)
operador!
inverte o resultado (de verdadeiro para falso e vice-versa)

p ~p
T F
F T
OR
p ~p
1 0
0 1

!True === False
!False === True
